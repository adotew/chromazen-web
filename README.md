# Chromazen Web

Minimal landing page for [Chromazen](https://github.com/adotew/minipaint-rs), built with Svelte 5, TypeScript, Vite, and WebGL2.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run check
npm run build
```

The production build is written to `dist/` and can be deployed directly to Vercel.
